# Celebrare Photo Gallery Web App

A responsive photo gallery application built with React, Vite, and Tailwind CSS. It fetches photos from the Picsum API, allows users to search by author, and lets users favourite photos (persisted via `localStorage`).

## Tech Stack
- **React 18** (Hooks: `useState`, `useEffect`, `useReducer`, `useMemo`, `useCallback`)
- **Vite** (Build tool & dev server)
- **Tailwind CSS v4** (Styling)

## How to Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
